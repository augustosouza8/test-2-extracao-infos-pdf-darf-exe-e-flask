================================================================================
                    EXTRATOR DARF - DESKTOP NATIVO
================================================================================

✅ IMPLEMENTAÇÃO COMPLETA!

A aplicação desktop nativa com PyQt6 está 100% implementada e pronta para 
gerar o executável .exe para distribuição.

================================================================================
COMO GERAR O EXECUTÁVEL (3 PASSOS SIMPLES)
================================================================================

OPÇÃO RÁPIDA (Recomendada):
  1. Execute: GERAR_EXE_COMPLETO.bat
  2. Aguarde finalizar (15-30 minutos)
  3. O executável estará em: dist\ExtratorDARF.exe

OPÇÃO MANUAL:
  1. Execute: setup_exe.bat          (instala dependências)
  2. Execute: build_exe.bat          (gera o executável)
  3. Execute: testar_exe.bat         (testa o executável)

================================================================================
O QUE FOI IMPLEMENTADO
================================================================================

✅ Interface desktop nativa com PyQt6
✅ Upload de PDFs (drag-and-drop e seleção)
✅ Processamento assíncrono (não trava)
✅ Geração de Excel com 3 abas
✅ Gerenciamento de regras (códigos e CNPJs)
✅ Banco de dados SQLite (compatível com versão web)
✅ Modelos OCR incluídos no executável
✅ Scripts de build automatizados

================================================================================
PROBLEMAS COMUNS
================================================================================

"PyQt6 não encontrado"
  → Execute: setup_exe.bat

"Modelos OCR não encontrados"
  → Execute: setup_exe.bat (baixa os modelos)

"Erro ao gerar executável"
  → Execute: verificar_antes_build.bat (verifica problemas)

"Executável não abre no Windows"
  → Verifique se Windows Defender não está bloqueando
  → Clique com botão direito → Propriedades → Desbloquear
